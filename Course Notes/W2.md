# Bioinformatics Course Notes - Second Lecture
## 1. Linguistics
* Sequence of letters -> English grammar
* Sequence of nucleotides -> Regular grammar for genomic language
    * Hidden Markov Model (`HMM`)
        * `GENSCAN` (find ex/introns)
        * `Pfam` (find pr families)
    * Language of RNA (coding/non-coding) -> dynamic base pairing (*trans*- / *cis*-)
        > *trans*- : double RNA interaction; *cis*- : folding of a single RNA
        * `SCFG` (2<sup>nd</sup> structure of RNA)
        * `Rfam`
    > Above: context-free; below: context-sensitive
    * `Transformer` (3<sup>rd</sup>/4<sup>th</sup> structure of RNA)
        * Self-attention: long distance connection
        * Position embed: fast with parallel computing
        > `Alphafold`
    * `MultiOmics` Studies -> `AIVC` (a multimodal AI model)
