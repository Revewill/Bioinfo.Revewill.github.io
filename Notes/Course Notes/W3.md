# Bioinformatics Lecture Notes - Third Lecture
* BLAST
> General approach to pairwise alignment:
> select an algorithm that generates a score, allow gaps, global or local, estimate probability that the alignment occurred by chance.
## 1. Scoring Matrix
* Scoring functions
  * *e.g.*, perfect match: +2; mismatch: +1; gap: 0
  * Alignment based on protein **codon** can be more informative than DNA seq alignment
    * Codon degeneracy
    * Different substitution rates of different pairs of aa
* Gap penalty
  * Penalty model: $W_n = a+(n-1)b$
  * Arguments $a$ and $b$ are acquired from fitting
## 2. Needleman-Wunsch Algorithm
* Global alignment algorithm
  * Recursive algorithm using DP
  * Selects the best scoring alignment
    * Construct alignment matrix
    * Put `1` on every match
    * Each cell's score = its original score + max (previous row, previous col)
    * Trace back
* Miss short, highly similar substrings out-weighed by the rest of the seq
## 3. Smith-Waterman Algoeithm
* Local alignment algorithm
  * Searches substrings of highest similarity
  * Useful for finding domains or substrings within seqs
    * The score in each cell is the max possible score for an alignment of any length ending at the position
## 4. BLAST: local
* Objective: given a `query` seq, find the best match from a `database`
* NW/SW: Computation time too much
  * Requires heuristic methods
* Heuristic methods
  * Hypothesis: seqs are likely to be similar if they have a sufficient number of short local matches (**hits**/**tuples**)
  * Pre-compute a look-up table (**hash table**), listing positions of every possible tuple
  * $k$-**tuples** are indexed in database
* BLAST looks for neighbothood words instead of perfect matches
  * Raw Score & bit score
  * E-value and P-value
    * P-value: probability that an event occurs by chance
    * E-value: correction of P-value based on multiple testing
  * Identities and positives
    * Identities: perfect match
    * Positives: match that is similarin chemical properties
## 5. Homology
* Similarity ≠ Homology
* Ortholog (frog α and mouse α), paralog (frog α and frog β)
* Pairwise alignment (BLAST) to multiple alignment (HMM, SCFG)
