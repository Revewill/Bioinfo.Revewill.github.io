Bioinformatics Course Notes - First Lecture

1 Lecture Notes: Big Data-Driven Bioinformatics

1.1 Core Framework: 4 Steps

  • Question: Start with a biological question (e.g., how does a certain gene change in cancer?)
  • Information: Acquire high-throughput sequencing data (RNA-seq, ChIP-seq, etc.); large public databases provide vast information.
  • Analysis: Data cleaning, feature extraction, applying statistical and machine learning methods.
  • Modeling: Build models to interpret data and make predictions (e.g., gene function prediction).

1.2 Evolution of Scientific Paradigms

  • Fourth Paradigm (Data-driven): Directly mine patterns from data, no longer relying solely on hypotheses.
  • Fifth Paradigm (AI-driven): Use deep learning to automatically learn complex patterns, accelerating new discoveries (e.g., AlphaFold).

1.3 Key AI Model Principles

  • Hidden Markov Model (HMM): Gene prediction.
  • Stochastic Context-Free Grammar (SCFG): RNA secondary structure prediction.
  • Transformer: Protein structure prediction and LLMs.

1.4 Algorithm vs Model

  • Algorithm: A set of concrete steps or methods to solve a problem; it is a general computational process.
    Example: DP algorithm, gradient descent algorithm.
  • Model: A parametric representation learned from data; it is the result of applying an algorithm to specific data.
    Example: trained HMM parameters, neural network weights.
